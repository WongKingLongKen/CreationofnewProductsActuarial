How to Use This Tool

  1. Upload a CSV file containing insurance product data
  2. Set the output directory where processed files will be saved
  3. Define the product code mappings (original to new codes)
  4. Review the file preview to ensure data is correctly loaded
  5. Click "Process CSV File" to execute the conversion
  6. Download the processed file from the output directory
